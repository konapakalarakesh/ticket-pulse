ℹ️ How This Dashboard Works
  • Authentication: Uses your browser's existing Midway session cookies — nothing stored externally.
  • Ticket Status Fetch: Calls t.corp.amazon.com directly using your authenticated browser session (same-origin credentials).
  • Audit Data: Loads your Audits.xlsx from GitHub (raw URL). Expects columns: Date, Process, Ticket ID, Auditor, Bucket.
  • Hosting: Pure static HTML — push to GitHub Pages, enable Pages, share the link.
  • Caching: Status results cached in browser localStorage (default 30 min) to reduce redundant fetches.
  • Privacy: All data processing happens in your browser. No external servers involved.
🚀 Setup Steps
  1. Create a GitHub repo and upload this index.html and your Audits.xlsx
  2. Go to repo Settings → Pages → Source: main branch → Save
  3. Copy the raw URL of your Excel: https://raw.githubusercontent.com/YOU/REPO/main/Audits.xlsx
  4. Paste it in the "Audit Excel URL" field above and click Save
  5. Share the GitHub Pages URL with your team
